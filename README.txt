# Roblox Tour Website

Upload `index.html` to GitHub.

To change the event info:
- Edit the text inside `<div class="status">`
- Edit the date inside `<div class="date-box">`
- Edit the city inside `<div class="city">`

The image in `/assets` is just the visual reference mockup.
The website itself is built in HTML/CSS so it still works without the image.
